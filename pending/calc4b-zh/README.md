## 下载

### Docker

```
docker pull apachecn0/calc4b-zh
docker run -tid -p <port>:80 apachecn0/calc4b-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install calc4b-zh
calc4b-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g calc4b-zh
calc4b-zh <port>
' + 
                '# 访问 http://localhost:{port} 查看文档
```
